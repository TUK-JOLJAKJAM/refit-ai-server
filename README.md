# 🏋️‍♂️ ReFit AI Analysis Server

**ReFit** 프로젝트의 핵심 엔진으로, 게임 기반 재활 훈련 중 발생하는 센서 데이터를 실시간으로 분석하여 의학적 지능형 피드백을 제공하는 AI 서버입니다.

## 🌟 프로젝트 개요
본 서버는 사용자(환자)가 재활 게임을 수행하는 동안 전달되는 16ms 주기의 고해상도 센서 데이터를 분석합니다. 단순한 게임 점수 계산을 넘어, 해부학적 관절 가동 범위(ROM)를 산출하고 움직임의 질(안정성)을 평가하여 맞춤형 재활 경로를 제시합니다.

## 🛠 주요 기능
- **실시간 관절 가동 범위(ROM) 산출**: 가속도/자이로 센서 데이터를 물리적 수식 및 알고리즘으로 처리하여 실시간 굴곡, 신전, 회전 각도 계산.
- **움직임 품질 및 안정성 분석**: 동작의 흔들림(Variance)과 속도를 분석하여 부적절한 운동 패턴 감지 및 피드백 생성.
- **동적 난이도 조절 (DDA)**: 수행 점수와 통증 수치를 결합하여 차기 세션의 운동 강도(목표 각도, 속도 등)를 자동 추천.
- **전문가 시스템(Expert System) 기반 로직**: 물리치료 데이터 시트 규격을 준수하는 규칙 기반 분석 엔진 구축.

## 🎮 분석 대상 게임 (6종)
1. **스트레칭 (Baseline)**: 전 부위 기초 ROM 측정 및 사용자 신체 데이터 수집.
2. **장작패기 (어깨)**: 전방굴곡 및 후방신전 동작의 각도와 속도 분석.
3. **몬스터타워 (하체)**: 스쿼트 깊이(각도) 및 버티기 시간 기반의 근지구력 평가.
4. **별자리 그리기 (손목)**: 손목 꺾기 및 회전 동작의 궤도 정확도 분석.
5. **훈련 정리 (손목)**: 손목 외회전/내회전 동작 수행 횟수 및 각도 측정.
6. **마을 지키기 (허리)**: 허리 회전 방향별 최대 각도 및 반응 속도 분석.

## 🏗 기술 스택
- **Language**: Python 3.10+
- **Framework**: FastAPI (High-performance Async Web Framework)
- **Library**: NumPy, Pandas (Data Analysis), Pydantic (Data Validation)
- **Architecture**: Modular Expert System for Biomechanical Analysis
